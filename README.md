# TempMonitor

一款专为 Windows 11 打造的、像素级精致的系统监控挂件。

---

## 🎯 核心亮点

*   **原生体验**：深度集成 Windows 11 视觉风格，采用 60% 透明度深空灰磨砂背景与 1px 微光白边，支持边缘磁吸吸附。
*   **极致性能**：基于 Win32 API (P/Invoke) 和 PerformanceCounter，实现 0 延迟数据采集，内存占用极低。
*   **真实数据**：物理级内存探测算法（拒绝虚拟内存虚标）+ 显卡专用显存 (Dedicated VRAM) 精确过滤。
*   **动态交互**：
    *   **智能红绿灯**：根据负载动态切换颜色（白色正常、橙色预警、红色高载）。
    *   **悬浮自动扩展**：鼠标悬停即平滑展开查看历史最高峰值 (MAX)，并具备避让边缘的智能生长方向。
*   **开机即用**：单文件绿色版，无环境依赖，支持一键设置开机自启。

## 🔍 交互指南

*   **磁吸吸附**：将窗口拖动到屏幕左右边缘 50px 范围内，松手即可自动贴合。
*   **智能展开**：
    *   在左侧或中间时：向右侧展开 MAX 数据。
    *   在右边缘吸附时：自动识别并向左侧“生长”，确保内容不溢出屏幕。
*   **快捷控制**：
    *   **点击 CPU 行**：瞬间启动 **Windows 任务管理器**。
    *   **双击窗口/右键菜单**：快速退出程序。
    *   **右键菜单**：重置历史最大值、切换开机自启状态。

## 🛠️ 技术栈 (Under the Hood)

*   **框架**：.NET 10.0 (WPF) - 采用最新的 .NET 技术栈。
*   **核心库**：`LibreHardwareMonitor` - 深度定制的 Visitor Pattern 实现，确保传感器数据的深度刷新。
*   **内存技术**：**Win32 P/Invoke (`GlobalMemoryStatusEx`)** - 绕过托管层，直接读取 OS 物理内存结构，保证 100% 准确性。
*   **指标采集**：`PerformanceCounter` - 利用 Windows 内核计数器，以极低的 CPU 开销获取实时负载。
*   **动画系统**：WPF Storyboard + DoubleAnimation - 实现 0 掉帧的平滑交互过渡。
*   **持久化**：JSON 序列化配置 + 屏幕边界自适应算法，确保窗口位置永不丢失。
